# Draft Cover Letter

Dear Editors-in-Chief,

We submit the manuscript **"DAIMO: An Ontology Profile for Governed AI Model Assets in Dataspaces"** for consideration in *Semantic Web - Interoperability, Usability, Applicability*.

The manuscript presents DAIMO, an OWL 2 DL integration profile for representing AI models as governed assets in dataspaces. DAIMO connects MLDCAT-AP, DCAT, ODRL, PROV-O, and the Dataspace Protocol to make publication, discovery, invocation, execution traceability, audit evidence, and contextualised evaluation queryable and validatable across organisational boundaries.

The contribution is suitable for the journal because it addresses a Semantic Web interoperability problem through explicit ontology reuse, axiomatised alignments, SHACL validation, competency questions, and a reproducible artefact package. The paper includes:

- a reuse-first ontology profile with 14 DAIMO-native classes;
- explicit alignment to established vocabularies;
- 23 competency questions implemented as SPARQL queries;
- SHACL structural validation and six cross-class SHACL-SPARQL invariants;
- a negative-test harness showing that each invariant catches its intended violation;
- OWL reasoning checks with HermiT and OWL-RL;
- an anonymised multi-participant demonstrator;
- public release metadata and reproducibility instructions.

We suggest handling the manuscript as an **Original Research Article** with a substantial ontology-engineering contribution. If the editorial team considers it more appropriate, it can also be evaluated as a description of an ontology artefact.

The artefact package will be made available through a stable public repository and Zenodo DOI before final submission. The current release candidate contains the ontology source, SHACL shapes, example graph, SPARQL queries, validation scripts, generated serialisations, validation reports, and replay instructions.

The manuscript is original, is not under review elsewhere, and all listed authors have approved the submission.
